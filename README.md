# GameDrop

Desktop game deal notifications. Real-time alerts for free games and discounts across Steam, Epic, GOG, and more.

## Features

- Real-time deal notifications via system tray
- Multi-provider: Steam, Epic Games, GOG, CheapShark, GamerPower
- Offline cache — deals stay visible without internet
- Filter by store, type, and recency
- Auto-start with Windows
- English & Bahasa Indonesia
